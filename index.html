Homebrew

El manejador de paquetes perdido para OS X

  • Homebrew es la forma más fácil y flexible de instalar las herramientas de UNIX que Apple no incluye en OS X.

    $ brew install wget

  • Los paquetes son instalados en su propio prefijo aislado y luego enlazados a /usr/local.

    $ cd /usr/local
$ find Cellar
Cellar/wget/1.12
Cellar/wget/1.12/bin/wget
Cellar/wget/1.12/share/man/man1/wget.1

$ ls -l bin
bin/wget -> ../Cellar/wget/1.12/bin/wget

  • Solo extrae el tarball y sin más ya tienes una instalación utilizable de Homebrew.

    ¡Instala Homebrew hoy mismo!

  • Crea nuevos paquetes de Homebrew en segundos.

    $ brew create http://foo.com/bar-1.0.tgz
Created /usr/local/Library/Formula/bar.rb

  • Adapta fácilmente una fórmula Homebrew A tus necesidades. Y ya que todo funciona con Git por debajo tus cambios son fusionados automáticamente con las actualizaciones principales.

    $ brew edit wget # opens in TextMate!

  • Las fórmulas Homebrew son simples scripts Ruby:

    require 'formula'

class Wget < Formula
  homepage 'http://www.gnu.org/wget/'
  url 'http://ftp.gnu.org/wget-1.12.tar.gz'
  md5 '308a5476fc096a8a525d07279a6f6aa3'

  def install
    system "./configure --prefix=#{prefix}"
    system 'make install'
  end
end

  • Homebrew complementa OS X. Instala tus gems con gem y sus dependencias con brew.
